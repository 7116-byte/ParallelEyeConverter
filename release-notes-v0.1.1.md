# v0.1.1

修复启动和辨识度问题。

- 修复新安卓版本录屏授权后可能停在待转化 App、没有画面反馈的问题。
- 注册 `MediaProjection.Callback`，避免新系统拒绝创建捕获面。
- 授权后默认显示红色悬浮球，进入目标 App 后双击悬浮球打开平行眼画面。
- 平行眼画面增加控制条：`-` 最小化回悬浮球，绿色 `X` 直接停止转化。
- 重做主界面为深色控制台风格，不再使用默认系统按钮外观。
- 增加应用图标。
- APK 输出文件改为 `ParallelEyeConverter-v0.1.1-debug.apk`，避免和其他项目的 `app-debug.apk` 混淆。
